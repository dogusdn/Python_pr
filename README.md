### 숫자 처리 함수
```python

print(pow(4, 2)) # 4^2 = 4 * 4 = 16
print(max(5, 12)) # 12
print(min(5, 12)) # 5
print(round(3.14)) # 반올림 값 3
print(round(4.99))  # 5
````
